# eeg-ecg
所使用的分法是與醫生各項評估後決定使用。

ECG

心跳是以R波作為擷取的方法。
RRI特殊是正常^^。

HRV

Heart rate variability，心率變異分析
使用時域分析指標RMSSD(Root Mean Square of Successive Differences)與SDNN(Standard Deviation of NN intervals)。

EEG

腦波則是使用樣本熵（SampEn）作為主要分析方法。
特別注意雜訊干擾

EEG與ECG尚未整理完畢。
